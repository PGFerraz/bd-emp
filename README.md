# Sistema de Gerenciamento de Pedidos
<img width="1401" height="896" alt="Image" src="https://github.com/user-attachments/assets/6ee54040-4931-434e-acac-853b4eabaaa9" />

Um sistema para **cadastro, organização e consulta de pedidos**, desenvolvido em **Python + PySide6** com banco de dados **MongoDB**.  
Ideal para qualquer área que precise registrar e acompanhar pedidos, como serviços, vendas, suporte técnico, projetos e muito mais.

---

## Funcionalidades

### Cadastro de Pedidos
- Dados completos do cliente  
- Data, endereço e telefone  
- Descrição detalhada  
- Forma de pagamento: **à vista** ou **parcelado**  
- Valor total  
- Parceiros associados  
- Status (pendente / finalizado)

### Busca Avançada
- Pesquise por:
  - nome do cliente  
  - número do pedido  
  - trechos da **descrição**
- Resultados organizados e completos

### Gestão de Parceiros
- Visualização de todos os parceiros cadastrados  
- Filtro por nome  
- Conexão direta com seus respectivos pedidos
